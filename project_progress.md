## 현재 완료
- 운영 구조를 `GitHub Pages + static JSON + GitHub Actions`로 정리했다.
- 서울 기준 1365 공고를 수집하고 [docs/data/volunteer_posts.json](/c:/Users/BJEon/Desktop/1365-vols-pages/docs/data/volunteer_posts.json)에 저장하는 배치를 만들었다.
- 모집 인원, 신청 인원은 상세 페이지 보강으로 채우고, 상세 캐시도 붙였다.
- 검색 화면은 정적 페이지 1개로 정리했고 초기 진입 시 전체 공고가 보이도록 맞췄다.
- 검색 조건은 시/도, 시/군/구, 키워드, 최소 모집인원, 봉사기간, 정렬, 페이지 크기까지 반영했다.
- 제목 클릭 시 상세 모달이 열리고, 원문 링크와 유사 공고 추천을 볼 수 있게 했다.
- 유사 공고 추천은 로컬 임베딩 기반 배치로 `similar_post_ids`를 JSON에 저장하는 방식으로 구현했다.
- 유사 공고는 현재 검색 조건을 유지한 상태에서 다시 필터링해 보여주도록 맞췄다.
- 상세 HTML에서 본문(`description`)도 함께 파싱하도록 바꿨다.
- LangChain + OpenAI 기반 `summary`, `tags` 생성 배치 초안을 만들고 병렬 처리까지 붙였다.
